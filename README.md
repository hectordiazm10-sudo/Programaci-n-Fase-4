# Programación Fase 4 – Software FJ

## Descripción

Sistema integral de gestión de clientes, servicios y reservas
desarrollado en Python para la empresa Software FJ.

El proyecto fue desarrollado aplicando programación orientada
a objetos y manejo avanzado de excepciones.

## Tecnologías utilizadas

- Python
- Programación orientada a objetos
- GitHub

## Funcionalidades

- Registro de clientes
- Gestión de servicios
- Reservas
- Manejo de excepciones
- Registro de errores

## Archivos del proyecto

- cliente.py
- servicio.py
- reserva.py
- excepciones.py
- main.py
- logs.txt
